Data redistribution attribution

Basis Set Exchange (CC-BY-4.0)

The numerical basis-set values shown in basis.html validation tables (STO-3G, STO-6G, 6-31G, 6-31G(d), 6-311G, def2-SVP exponents and contraction coefficients) originate from the Basis Set Exchange and are therefore a redistribution of CC-BY-4.0 data. Attribution is given via the "Basis Set Exchange (BSE)" entry in Sources, linking to basissetexchange.org, and via the primary publication citations in the "Basis sets" section of the same page.

The BSE web application itself (the Python code) is BSD-3-Clause; only the data values are CC-BY-4.0.

Public-domain data

Numerical values from CODATA 2022 (constants), IUPAC / CIAAW 2024 (atomic masses), and NIST CCCBDB (molecular geometries) are published by government-funded bodies and are in the public domain or freely reproducible with attribution. Attribution is given through Sources; no license text applies.
